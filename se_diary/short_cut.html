キーボード操作 on Mac　(リファレンス・逆引き)

1/21 2021 




    最初に覚えるべきショートカット キー


まず覚えるべきショートカット は

目的のアプリにまでたどり着くためのショートカットキー

です！

なぜならば、これらのショートカットキーは操作が単純で覚えやすい上に、
ショートカットを覚えた効果が一番はっきりと目に見えてわかるからです！

それでは早速行ってみましょう！





開いているウィンドウを切り替えるショートカット 


mac : [Cmd] + [Shift]
windows:[Alt] + [Shift]

これらのショートカット は使用しているアプリを時系列順に遡ることができます。

例えば、好きなパン屋さんの住所をChromeで調べて、それをexcelに書き込みたいとき...
Chromeをみて住所を確認して、それをexcelに移動して書き込む、
住所を忘れたから、また遡って確認して...

というアプリの往復移動を,

上のショートカット キー　一発で済みます！



    

アプリを検索する
    

タイピングスピードが早い人はまず最初にアプリを検索するショートカットを覚えましょう


mac: [Cmd] + Space => アプリの検索
windows: [Windows] => アプリの検索





    アプリの終了
        

アプリを開く方法、移動する方法がわかったら今度は終了する方法を覚えましょう！

[Cmd] + w 　=> アプリの終了


また、ポップアップウィンドウが邪魔だなーと思ったとき！

[Esc] => ポップアップウィンドの終了






    カーソルの選択範囲（入力フォーム）を動かすためのショートカット 


Tab => 時計回りで入力フォーム間を移動できる！
Shift + Tab => 反時計回りで入力フォーム間を移動！
